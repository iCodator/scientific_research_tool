    """

    These are search terms with database field codes attached.
    
    WHY IT MATTERS:
    PubMed and other databases allow searching specific fields.
    This function identifies such patterns for proper handling.
    
    PARAMETERS:
    token (str): A token from the parsed query
                 Example: "cancer"[MeSH]
    
    RETURNS:
    bool: True if token matches field-term pattern, False otherwise
    
    EXAMPLES:
